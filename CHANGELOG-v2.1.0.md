## ✨ 核心新功能

### 1. 多平台视频直链支持

现在外部直链功能支持多种视频平台，不仅仅是普通的 HTTP 直链！

#### 支持的平台类型

- **🔗 直链** - 普通 HTTP 视频直链（mp4/mkv/m3u8 等格式）
- **📺 Bilibili** - B站视频链接（支持 BV 号和 AV 号）
- **🌐 其他平台** - 其他视频平台的直链

#### 使用方法

1. 在后台管理进入"外部直链"页面
2. 点击"＋ 添加"按钮
3. 在弹窗中选择"平台类型"：
   - 选择"直链"用于普通视频链接
   - 选择"Bilibili"用于B站视频
   - 选择"其他平台"用于其他来源
4. 填写作品信息和视频链接
5. 保存即可

---

### 2. 封面图上传功能

现在可以为外部直链媒体上传本地封面图，不再局限于填写在线图片链接！

#### 功能特点

✅ **本地上传** - 从电脑选择图片文件直接上传  
✅ **实时预览** - 上传前即时预览封面效果  
✅ **自动存储** - 封面图自动保存到 R2 存储  
✅ **智能命名** - 按媒体ID和时间戳自动命名，避免冲突  
✅ **格式支持** - 支持 JPG/PNG/GIF/WebP 等常见图片格式  
✅ **大小限制** - 单文件最大 10MB，确保存储效率  

#### 存储结构

封面图存储在 R2 的专用目录：

```
_admin/covers/
└── {媒体ID}/
    └── {时间戳}.{格式}
```

**示例**：
```
_admin/covers/movies_星际穿越/1714435200000.jpg
_admin/covers/tvshows_权力的游戏/1714435300000.png
```

#### 使用方法

**方式一：上传本地图片**
1. 在添加/编辑外部直链时
2. 找到"封面图"字段
3. 点击"上传"按钮
4. 选择本地图片文件
5. 系统自动上传并填入URL

**方式二：填写在线链接**
1. 在"封面图"输入框中
2. 直接粘贴图片的 HTTPS 链接
3. 保存即可

外部直链列表现在显示平台类型徽章，一目了然！

#### 徽章样式

- 📺 **B站** - 粉色边框，显示"B站"标识
- 🔗 **直链** - 绿色边框，显示"直链"标识

- 加载失败自动隐藏

## 🔧 技术实现

### 新增 API 接口

**`/api/admin/upload-cover`** - 封面图上传接口

**请求方式**: POST (FormData)

**请求参数**:
- `cover`: 图片文件 (File)
- `mediaId`: 媒体ID (String)

**响应示例**:
```json
{
  "ok": true,
  "url": "/api/admin/download?key=_admin/covers/...",
  "key": "_admin/covers/movies_xxx/123456.jpg",
  "size": 123456
}
```

**安全特性**:
- ✅ HMAC-SHA256 身份验证
- ✅ 文件类型校验（仅图片）
- ✅ 文件大小限制（10MB）
- ✅ 路径安全过滤（防注入）

### 数据结构升级

外部直链数据新增 `platform` 字段：

```json
{
  "id": "movies/星际穿越",
  "type": "movies",
  "name": "星际穿越",
  "platform": "bilibili",        // 新增：平台类型
  "year": "2014",
  "poster": "/api/admin/download?key=...",
  "url": "https://...",
  "episodes": [],
  "source": "external"
}
```

**platform 字段值**:
- `"direct"` - 普通直链（默认值，向后兼容）
- `"bilibili"` - B站视频
- `"other"` - 其他平台

## 📖 使用场景

### 场景一：添加B站视频到媒体库

1. 复制B站视频链接（如 `https://www.bilibili.com/video/BV1xx4y1c7mQ`）
2. 在后台添加外部直链
3. 平台类型选择"Bilibili"
4. 填写作品名称（如"流浪地球2"）
5. 上传或填写封面图
6. 粘贴B站视频链接
7. 保存

### 场景二：为现有直链更新封面

1. 在外部直链列表找到目标项目
2. 点击"编辑"按钮
3. 点击"上传"按钮选择新封面
4. 预览确认效果
5. 保存更新

### 场景三：批量管理多平台内容

- 使用来源筛选（全部来源/手动外链/Alist导入）
- 使用平台徽章快速识别不同来源
- 封面缩略图辅助视觉识别

## 🔄 向后兼容性

✅ **完全兼容旧版本数据**
- 旧的外部直链自动识别为"直链"平台
- `platform` 字段默认为 `"direct"`
- 无需手动迁移数据

✅ **保留原有功能**
- 仍支持填写在线图片链接
- 外部直链数据结构向后兼容
- 前台显示逻辑不受影响

## 📝 注意事项

1. **封面图大小**: 建议上传 500x700 像素左右的图片，过大自动压缩
