# Kingsley Media 播放器功能说明

## 🎬 播放器控制栏功能

### 基础控制

| 按钮 | 功能 | 说明 |
|------|------|------|
| ▶️ / ⏸️ | 播放/暂停 | 切换视频播放状态 |
| ⏪ | 快退10秒 | 向后跳转10秒 |
| ⏩ | 快进10秒 | 向前跳转10秒 |
| 🔊 / 🔇 | 音量/静音 | 切换静音状态 |
| ☀️ | 亮度调整 | 显示亮度滑块（50%-150%） |
| 📋 | 画中画 | 开启/关闭画中画模式 |
| ⛶ | 全屏 | 切换全屏播放 |

### 进度条

- **点击跳转**：点击进度条任意位置跳转到对应时间
- **拖拽调整**：按住进度条拖拽到目标位置
- **触屏支持**：支持触摸设备的拖拽操作
- **实时更新**：播放时实时显示进度

### 音量控制

- **滑块调节**：鼠标悬停在音量按钮上显示滑块
- **范围**：0% - 100%
- **静音切换**：点击音量按钮快速静音/恢复

### 亮度调整

- **点击太阳图标**：显示亮度滑块
- **调节范围**：50% - 150%
- **实时预览**：拖动滑块即时看到效果
- **自动隐藏**：点击其他地方自动隐藏滑块

### 画中画模式

- **独立窗口**：视频在小窗口中继续播放
- **多任务**：可以同时进行其他操作
- **浏览器支持**：需要浏览器支持 Picture-in-Picture API

## 🌐 外部直链播放

### 支持的平台

- **普通直链**：直接的视频/音频文件链接
- **Bilibili**：B站视频（需要特殊处理）
- **其他平台**：提供直接播放链接的平台

### URL 处理规则

1. **完整URL**（http://或https://开头）
   - 直接使用，不做修改
   - 示例：`https://example.com/video.mp4`

2. **API代理路径**（/api/开头）
   - 使用同域代理
   - 示例：`/api/alist/stream?...`

3. **相对路径**
   - 自动添加 CDN 前缀
   - 示例：`media/movies/xxx.mp4` → `https://cdn.kingsleyqi.com/media/movies/xxx.mp4`

### 跨域处理

- 播放器设置 `crossorigin="anonymous"` 属性
- 支持 CORS 的直链可以正常播放
- 不支持 CORS 的链接可能需要代理

## 💡 使用技巧

### 1. 快速跳转

- **大段跳转**：使用进度条点击/拖拽
- **微调位置**：使用 J/L 键或快进快退按钮

### 2. 夜间观看

- 点击亮度图标
- 将亮度调整为 50%-70%
- 减少眼睛疲劳

### 3. 多任务处理

- 点击画中画按钮
- 视频在小窗口继续播放
- 可以浏览其他内容或工作

### 4. 沉浸观看

- 按 F 键进入全屏

### 5. 进度保存

- 播放进度自动保存
- 下次播放从上次位置继续
- 进度存储在浏览器本地

## ⚠️ 常见问题

### 视频无法播放

**可能原因：**
1. 格式不兼容（如 MKV 在某些浏览器）
2. 跨域限制（CORS 问题）
3. 链接已失效
4. 网络问题

**解决方法：**
- 查看浏览器控制台错误信息
- 尝试转码为 MP4 (H.264/AAC)
- 使用 Alist 代理播放
- 检查网络连接

### 没有声音

**检查项：**
1. 是否静音（点击音量图标）
2. 系统音量设置
3. 浏览器标签页是否静音
4. 视频文件本身是否有音轨

### 卡顿缓冲

**优化建议：**
- 检查网络速度
- 降低视频清晰度（如果是 HLS）
- 等待缓冲后再播放
- 使用代理服务器加速

### 亮度调整无效

**说明：**
- 亮度调整是通过 CSS filter 实现
- 只在播放器容器内生效
- 不影响系统或其他页面亮度
